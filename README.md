### 기능 소개
#### 1. 캐릭터 체크
- 보유 캐릭터를 직접 체크하여 보유 상태를 기록합니다.  

#### 2. 결과 분석
- 명함도 없는 캐릭터, 클래스 체인지 가능 캐릭터 등 여러 결과를 확인할 수 있습니다.
